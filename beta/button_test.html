Beta

Status

Live state of both banner buttons.

Fullscreen: off

Usage

Add these two lines to any published app's <head> and insert the button next to the theme toggle.

<link rel="stylesheet"
  href="/assets/fullscreen-button.css">
<script src=
  "/assets/fullscreen-button.js"></script>

HTML

Button element — place it after the theme toggle inside .banner-actions.

<button
  id="fullscreen-button"
  data-panphy-fullscreen
  aria-label="Enter fullscreen"
  title="Enter fullscreen">
</button>

Target element

To fullscreen a specific element instead of the whole page, set the attribute value to a CSS selector.

data-panphy-fullscreen="#workspace"
