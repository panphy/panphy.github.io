Quick Math.js Reference

This page is a practical cheat-sheet for formulas used in PanPhyPlot Data Processing. It is aimed at pre-university students, so it focuses on common operators and functions you are most likely to use.

Quick Start (In Data Processing)

  • Use variables x and y in formulas.
  • You can also use valid custom header names as variables.
  • Type formulas like =1/x, x^2, log(y), then press Enter or click Apply.
  • Use lowercase constants: pi and e.

Operators

Operator Meaning Example
+ Add x + y
- Subtract / negative sign x - y, -x
* Multiply 2*x
/ Divide y/x
^ Power x^2, y^0.5
% Remainder (mod) x % 2
( ) Group parts of an expression (x + y)/2

Common Functions

Trig (angles in radians)

Function Meaning Example
sin(x) Sine sin(x)
cos(x) Cosine cos(x)
tan(x) Tangent tan(x)
asin(x), acos(x), atan(x) Inverse trig atan(y/x)

Powers, Roots, and Absolute Value

Function Meaning Example
sqrt(x) Square root sqrt(x)
cbrt(x) Cube root cbrt(y)
abs(x) Absolute value abs(y)

Logs and Exponentials

Function Meaning Example
log(x) Natural log (base e) log(y)
log10(x) Base-10 log log10(y)
exp(x) e^x exp(x)

Rounding

Function Meaning Example
round(x) Round to nearest integer round(3.6) gives 4
floor(x) Round down floor(3.9) gives 3
ceil(x) Round up ceil(3.1) gives 4

Constants

Constant Value (approx.) Example
pi 3.14159... 2*pi*x
e 2.71828... e^x or exp(x)
tau 6.28318... (2*pi) tau*x
Infinity Very large positive value marker Usually not needed in school-level transformations

Tip: Math.js is case-sensitive.

Use pi and e in lowercase. Typing PI or E may be treated as unknown variables.

Useful Formula Ideas

  • =1/x : reciprocal transform
  • =x^2 : square the x-data
  • =sqrt(y) : square-root transform of y
  • =log(y) : natural log transform
  • =sin(2*pi*x) : sinusoidal model input
  • =(x+y)/2 : average of x and y
  • =abs(y) : convert y to magnitude only

This is a quick reference, not the full Math.js documentation. If a formula gives an unknown-variable warning, check spelling and capitalization first.
